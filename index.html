GarticTools — Educational WebSocket Analysis Suite

🎯 Purpose

This project analyzes how Gartic.io uses WebSockets for real-time communication. It's built for learning, research, and technical curiosity.

Goal: Understand game behavior to improve quality — ethically and legally.
