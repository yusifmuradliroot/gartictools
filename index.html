Windows Təlimatları

×
  1. Gartic.io saytını brauzerinizdə açın (Chrome, Firefox və ya Edge)
  2. Sağ klik edib "Səhifə məlumatı" > "Brauzer rejimi" seçin
  3. "Desktop" rejimini seçin (mobil rejim deyil)
  4. Ctrl + Shift + I düymələrini basın (və ya F12) və "Console" (Konsol) bölməsinə keçin
  5. Aşağıdakı kodu konsola daxil edin və Enter basın:
// Söhbət funksiyasını düzəldən skript
if (typeof fixChat === 'undefined') {
  var fixChat = function() {
    console.log('Gartic-Chat-Fix aktivləşdirildi!');
    // Söhbət elementlərini düzəldən kod burada olacaq
    document.querySelector('.chat-input').style.bottom = '70px';
    document.querySelector('.chat-container').style.bottom = '120px';
  };
  window.addEventListener('load', fixChat);
}
  1. Saytı yenidən yükləyin (F5 düyməsi)
  2. Söhbət indi düzgün işləməlidir
