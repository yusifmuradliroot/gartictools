💬 Gartic-Chat-Fix

Bu alət Gartic.io-da söhbət problemlərini həll etməyə kömək edir. Aşağıdakı düyməyə basaraq başlaya bilərsiniz.

Gartic-Chat-Fix

×

Hansı əməliyyat sistemini istifadə edirsiniz?

Android Təlimatları

×
  1. Əvvəlcə Gartic.io saytına brauzerinizdən daxil olun
  2. Sağ yuxarı küncdəki üç nöqtəyə (⋮) vurun
  3. "Desktop sayt kimi aç" seçin
  4. Saytı yenidən yükləyin (yenidən açın)
  5. Söhbət xanasına vurduqda klaviaturanın düzgün göstərilməsi üçün ekranı yana çevirin
  6. Əgər problem davam edirsə, brauzerinizin keşini təmizləyin
  7. Ən yaxşı nəticə üçün Chrome və ya Firefox istifadə edin
Diqqət: Bu təlimatlar yalnız təlim məqsədləri üçündür. Oyunda hər hansı pozuntu yaratmaq qadağandır.

Windows Təlimatları

×
  1. Gartic.io saytını brauzerinizdə açın (Chrome, Firefox və ya Edge)
  2. Sağ klik edib "Səhifə məlumatı" > "Brauzer rejimi" seçin
  3. "Desktop" rejimini seçin (mobil rejim deyil)
  4. Ctrl + Shift + I düymələrini basın (və ya F12) və "Console" (Konsol) bölməsinə keçin
  5. Aşağıdakı kodu konsola daxil edin və Enter basın:
// Söhbət funksiyasını düzəldən skript
if (typeof fixChat === 'undefined') {
  var fixChat = function() {
    console.log('Gartic-Chat-Fix aktivləşdirildi!');
    // Söhbət elementlərini düzəldən kod burada olacaq
    document.querySelector('.chat-input').style.bottom = '70px';
    document.querySelector('.chat-container').style.bottom = '120px';
  };
  window.addEventListener('load', fixChat);
}
  1. Saytı yenidən yükləyin (F5 düyməsi)
  2. Söhbət indi düzgün işləməlidir
Diqqət: Bu təlimatlar yalnız təlim məqsədləri üçündür. Oyunda hər hansı pozuntu yaratmaq qadağandır.
